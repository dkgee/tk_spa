
# Puppeteer实践

## NPM命令
	1、npm init --yes	初始化目录，执行后当前目录会出现package.json
	1、npm install xxx	安装依赖模块

## 依赖模块说明
	1、puppeteer：Chrome的无界面版本及对其进行操控的js接口套装
	2、crypto：js加密模块
